# Password Strength Checker

A command-line Python script that evaluates the strength of a password.

## How it works

- Asks the user for a password.
- Checks if the password is at least 8 characters long.
- Checks if the password contains any special symbols.
- Outputs a strength rating: **Weak**, **Medium**, or **Strong**.

## How to run

1. Make sure you have Python 3 installed.
2. Download the script or clone the repository.
3. Run the script:
   ```bash
   python password_checker.py
